# Courses Published in Q3

In order to find out which preservation/dissemination skill is more relevant to the community, we decided to develop several series of mini-courses covering every stage texts (in the modern sense of the term). The start with the creation of digital texts (scanning, curating images and sound tracks, transcription, OCR), curation (using librarian software for cataloging, organization, publication), editing (proofreading, creation of classic epubs, creation of interactive epubs with video overlay), dissemination (contributing texts to public platforms), general usage (search, browsing, searching for specific information, generating citations), and specialized usage (etexts in professional translation CAT tools, using etexts and statistics to teach language, using etexts and statistics to teach traditional subjects).

As soon as we have a clearer idea of the courses that are the most in demand we will keep developing extensive format courses on BudaX, for instance, since we started to publish the mini-courses it has become clear that tutorials for the BDRC Lib mobile app don’t require a full-fledged course on BudaX, but are a perfect fit for the mobile-friendly mini-course format. We also got requests to expand on the course on proofreading with regular expression in Visual Studio Code and SarAnt, which will probably require a more in-depth coverage in (10 days long?) BudaX course.

During Q3, we’ve published mini-courses in 3 series, “Pecha Reader of the 21st Century”, “Proofreader of the 21st Century”, and “Scribe of the 21st Century”.
