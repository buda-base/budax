# Code软件的一般应用方式 སྤྱིར་བཏང་ཁོ་ཌི་མཉེན་ཆས་བཀོལ་སྟངས། 

这里讲述一般Visual Studio Code或者Code软件的总体应用和以下两种搜索方式 སྤྱིར་བཏང་གི་ Visual Studio Code འམ་ཁོ་ཌི་མཉེན་ཆས་བཀོལ་སྤྱོད་བྱེད་སྟངས་སྤྱི་དང་ལྷག་པར་དུ་གཤམ་བཀོད་ཀྱི་བཙལ་བྱ་རིགས་འཚོལ་སྟངས་ཀྱི་རིམ་པ་གཉིས་བཅས་ངོ་སྤྲོད་བྱས་ཡོད། 

🔊 提示：此软件上只能应用文本文档(.text)，其他格式不可应用。 གསལ་འདེབས། མཉེན་ཆས་འདིའི་ནང་ཡིག་རྐྱང་(.txt)མ་གཏོགས་གཞན་བཀོལ་མི་ཐུབ། 
## Code软件的基本特点 ཁོ་ཌི་མཉེན་ཆས་ཀྱི་ཁྱད་ཆོས་ཪགས་བསྡུས།  

👉 制造软件者的首选应用 མཉེན་ཆས་བཟོ་མཁན་གྱི་བཀོལ་ཆས་ཨང་དང་བོ།   
👉 最佳编辑文档器 ཡིག་ཆ་རྩོམ་སྒྲིག་བྱེད་ས་ལེགས་ཤོས།

👉 ལུང་ཚིག་འཚོལ་ཞིབ་བྱེད་ས་ལེགས་ཤོས།
